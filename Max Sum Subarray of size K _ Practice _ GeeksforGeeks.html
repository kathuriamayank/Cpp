Max Sum Subarray of size K

EasyAccuracy: 49.6%Submissions: 129K+Points: 2

Win from a cash prize pool of INR 15,000
Check out Hack-For-Health Hackathon Now!

Given an array of integers Arr of size N and a number K. Return the maximum sum of a subarray of size K.

NOTE*: A subarray is a contiguous part of any given array.

Example 1:

Input:
N = 4, K = 2
Arr = [100, 200, 300, 400]
Output:
700
Explanation:
Arr3  + Arr4 =700,
which is maximum.

Example 2:

Input:
N = 4, K = 4
Arr = [100, 200, 300, 400]
Output:
1000
Explanation:
Arr1 + Arr2 + Arr3 + Arr4 =1000,
which is maximum.

Your Task:

You don't need to read input or print anything. Your task is to complete the function maximumSumSubarray() which takes the integer K, vector Arr with size N, containing the elements of the array and returns the maximum sum of a subarray of size K.

Expected Time Complexity: O(N)
Expected Auxiliary Space: O(1)

Constraints:
1 <= N <= 105
1 <= Arri <= 105
1 <= K <= N

            sum += Arr.get(right);

            right++;
            if(right - left == K){
                sum -= Arr.get(left);
                left++;
            }
        }
        
        return max;
    }

C++ SOLUTION ||ALL TESTS CASES PASSED || SIMPLE UNDERSTANDABLE EXPLANATION

Introduction: Welcome to the Max Sum Subarray Explorer! Imagine you have an array representing a sequence of numbers. Your mission is to find the subarray (a consecutive part of the array) with the maximum sum. Let's break down the steps.

Code Explained:

  1. Choosing the Starting Point:

    • Think of the array as a playground slide, and we start at the top by picking the first K elements as our initial sum.
     
    long long int sum = 0; // Our cumulative sum holder long long int maxi = -1; // The maximum sum we've seen so far for (int i = 0; i < K; i++) { sum += Arr[i]; // Adding numbers to our sum } maxi = sum; // Setting the initial highest sum

  2. Sliding and Calculating Fun:

    • Now, we start sliding through the array. For each step, we add a new element to our sum and subtract the one we left behind K steps ago. We're seeking the maximum sum subarray!
     
    for (int i = K; i < N; i++) { sum += Arr[i] - Arr[i - K]; // Adding a new element and subtracting the one we left behind maxi = max(maxi, sum); // Updating the highest sum }

  3. Result: Max Sum Subarray Found!

    • After sliding through the entire array, we have our maximum sum subarray.
     
    return maxi; // The maximum sum subarray is ready to be explored!

Analogy: Imagine you're sliding along a sequence of numbers, trying to find the section with the most "height" (sum). The code helps you keep track of the highest sum as you slide through.

Time Complexity:

  • Checking the starting sum at the top of the slide: O(K) (K is the size of the starting subarray).
  • Sliding through the entire array: O(N) (N is the total number of elements in the array).

Total Time Complexity: O(K) + O(N) = O(K + N)

Space Complexity:

  • We use a variable to hold the cumulative sum, making it constant space.

Total Space Complexity: O(1) (constant space)

Aditya Verma's Approach with Comments:

class Solution {
public:
    long maximumSumSubarray(int k, vector<int> &arr, int n) {
        // Initialize variables
        long sum = 0; // Variable to store the sum of the current subarray
        int i = 0, j = 0; // Pointers to mark the start and end of the window
        long maxi = INT_MIN; // Variable to store the maximum sum encountered

        // Loop through the array
        while (j < n) {
            sum += arr[j]; // Add the current element to the sum

            // Check if the window size is less than k
            if (j - i + 1 < k) {
                j++; // Expand the window by moving the end pointer
            }
            // Check if the window size is equal to k
            else if (j - i + 1 == k) {
                maxi = max(maxi, sum); // Update the maximum sum if the current sum is greater

                sum -= arr[i]; // Remove the contribution of the element at the start of the window

                i++; // Move the start pointer to slide the window
                j++; // Move the end pointer to slide the window
            }
        }

        return maxi; // Return the maximum sum encountered in any subarray of size k
    }
};
